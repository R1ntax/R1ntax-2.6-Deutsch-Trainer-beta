# Где хранить базу немецких слов с переводами

В этом проекте глобальная база слов читается из **Cloud Firestore**.

## Куда добавлять слова

Добавляйте документы в коллекцию:

`words`

Каждый документ = одно немецкое слово.

## Формат документа

```json
{
  "german": "der Apfel",
  "translation": {
    "English": "apple",
    "Russian": "яблоко",
    "French": "pomme",
    "Spanish": "manzana"
  }
}
```

Допустимо поле `translations` вместо `translation` (оба поддерживаются кодом).

---

## Как добавить через Firebase Console

1. Откройте Firebase Console → Firestore Database.
2. Создайте коллекцию `words`.
3. Добавьте документы с полями `german` (string) и `translation` (map).
4. Добавьте минимум **4 слова**, иначе приложение останется на локальном fallback-словаре.

---

## Как это работает в приложении

- При старте приложение вызывает загрузку `words` из Firestore.
- Если слов из БД достаточно и формат валиден — режимы карточек и дуэли используют БД.
- Если Firebase не настроен или данных мало — используется встроенный `BASE_WORDS` fallback.

## Пример набора (минимум)

- `der Apfel` → English: `apple`
- `das Haus` → English: `house`
- `das Wasser` → English: `water`
- `der Baum` → English: `tree`


## Уровни немецкого (A1-C2)

Для каждого слова добавляйте поле `level`:

- `A1`
- `A2`
- `B1`
- `B2`
- `C1`
- `C2`

Пример документа:

```json
{
  "german": "das Angebot",
  "level": "B1",
  "translation": {
    "English": "offer",
    "Russian": "предложение"
  }
}
```

Приложение фильтрует слова по выбранному уровню пользователя в настройках.

## Размер базы

Рекомендуемый объём коллекции `words`: **3000+ слов** (лучше 5000+).

- минимум для запуска: 4 валидных слова
- целевой production-объём: 3000+

## Разделение синонимов и антонимов

В интерфейсе это теперь отдельные игровые вкладки:
- `Синонимы`
- `Антонимы`

## Быстрый генератор 3000+ слов

Добавлен скрипт:

- `tools/generate_words_seed.js`

Команда:

```bash
node tools/generate_words_seed.js 3600
```

Она создаёт файл `words_seed_3600.json` с 3600 документами формата `german + level + translation` для массовой загрузки в Firestore.
